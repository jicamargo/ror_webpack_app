# 📖 "Hello world" project as one app with Webpack <a name="about-project"></a>

**This "Hello World"** project is a simple and lightweight web application built using Webpack. It serves as an excellent starting point for developers who want to explore the basics of using Webpack in their web development projects.
## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>
### Built With <a name="built-with"></a>

#### Tech Stack <a name="tech-stack"></a>

- Ruby on Rails
- PostgreSQL
- React
- Redux
- Webpack

#### Key Features <a name="key-features"></a>

- Minimalistic setup: This project focuses on the core concepts of Webpack, making it an ideal choice for beginners.

- Webpack configuration: Gain hands-on experience with configuring Webpack for bundling JavaScript and other assets.

- Quick start: Easily get started with this project to understand how Webpack simplifies the development workflow.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

Ruby (version 3.2)
Ruby on Rails (version 7.0)
PostgreSQL (version 15)
Node.js (version 16.11.1)


### Setup

1. Clone the repository:

`git clone https://github.com/jicamargo/ror_webpack_app.git`

2. Install the dependencies:

`bundle install`
`npm install`

3. Set up the database

`rails db:create`
`rails db:migrate`

4. Start the server in a bash terminal:

`bin/dev`

5. Open your web browser and navigate to: http://localhost:3000/ to access the application.
